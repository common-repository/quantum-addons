Quantum Addons is an Elementor addons plugin which has highly customizable addons.

== Description ==

You just have to activate this plugin and all available addons will be in the Elementor editor.

== List of addons ==

- Advance Slider ([documentation](https://github.com/abhy12/quantum-addons/blob/master/widgets/advance-slider/README.md))
- More soon
